# Val Clancy Pedro - Portfolio En Ligne

Ce projet est un site portfolio personnel développé en HTML5 et CSS3 pur. Il présente mon profil en tant que développeur web et stagiaire chez **Akieni Academy**, mes compétences clés, mon parcours de formation ainsi qu'un formulaire permettant de me contacter directement.

Le design est entièrement responsive (adapté aux mobiles et tablettes) et utilise une esthétique soignée basée sur les polices Google Fonts (_Ubuntu_ et _Roboto_).

## Demo

Visualisez une demo de ce projet sur le lien suivant :
<https://profile-perso.netlify.app>

## 🚀 Fonctionnalités

- **Section Présentation :** Un en-tête immersif mettant en avant mon profil et ma photo de présentation.
- **Compétences :** Une liste claire de mes domaines d'expertise (Front-End, Back-End, Database).
- **Parcours & Formations :** Une frise chronologique détaillée de mon cursus universitaire et professionnel entre le Congo et le Maroc.
- **Formulaire de Contact :** Un formulaire stylisé et interactif pour recevoir des messages.
- **Réseaux Sociaux :** Des liens dynamiques et animés dans le pied de page pour me suivre sur GitHub, LinkedIn, Instagram et Facebook.
- **Responsive Design :** Optimisation de l'affichage pour les écrans plus petits (Media Queries).

## 💻 Installation et Lancement Local

Pour visualiser et tester ce projet sur votre machine locale, aucune configuration complexe ou serveur n'est nécessaire. Suivez simplement ces étapes :

1. **Cloner le dépôt GitHub :**

   ```bash
   git clone https://github.com/valpedro59/perso-profile
   ```

2. **Accéder au dossier du projet :**

   ```bash
   cd perso-profile
   ```

3. **Ouvrir le projet :**
   Double-cliquez simplement sur le fichier `index.html` pour l'ouvrir dans votre navigateur web préféré, ou utilisez l'extension **Live Server** sur VS Code pour un rechargement automatique lors de vos modifications.
